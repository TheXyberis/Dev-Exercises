# Movie Form Project

This is a simple React project for the "Add Movie" form.

## Features

- Controlled input for movie title
- Dropdown select for genre
- Submit button outputs data to console
- Bootstrap styling
- Ready for offline usage (all assets local)

## How to run

1. Install dependencies: `npm install`
2. Start dev server: `npm run dev`
3. Open browser at `http://localhost:5173`
