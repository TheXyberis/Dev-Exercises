# Image Gallery Project

This is a simple React project for an **Image Gallery with categories**.  
It demonstrates filtering, state management, and conditional rendering in React.  

## Features

- Display images with categories: Flowers, Animals, Cars
- Controlled checkboxes to show/hide categories
- Dynamic download counter for each image
- Responsive layout with flexbox
- Minimal inline styles (margin, border-radius)
- Fully offline: images stored locally in `public/assets`
- Uses React `useState`, `map`, and `filter`
